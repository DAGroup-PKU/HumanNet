# HumanNet Website

This repository contains the public website for **HumanNet**, a
human-centric video dataset project from **PKU DAGroup** and
collaborators.

The site introduces the dataset, shows representative third-person and
egocentric video examples, presents project members, and links visitors
to the project's GitHub, arXiv page, and early-access waitlist.

## Website

The GitHub Pages version is intended to be served at:

```text
https://dagroup-pku.github.io/HumanNet/
```

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- HeroUI

## Development

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm run dev:web
```

Build the site:

```bash
npm run build
```

Build the static GitHub Pages version:

```bash
npm run build:pages
```

## Notes

This repository is for the HumanNet website only. Dataset files,
benchmarks, and paper materials are published separately through the
official HumanNet project channels.
